# 🧭 The Boredless Tourist

The Boredless Tourist is a simple Python-based recommendation engine that helps travelers discover attractions that match their interests in different cities around the world.

The application evaluates a traveler’s interests and suggests relevant venues, restaurants, museums, and historical destinations to help them explore a city at their own pace.

---

## 🚀 Features

- Stores multiple travel destinations
- Adds attractions with descriptive tags
- Matches traveler interests with attractions
- Generates personalized recommendations
- Easy-to-read and beginner-friendly Python code

---

## 🛠️ Technologies Used

- Python 3
- Core data structures (lists, functions)
- No external libraries required

---

## 📂 Project Structure

- Destinations stored as a list
- Attractions stored as a nested list
- Functions to:
  - Add attractions
  - Search attractions by interest
  - Generate personalized travel suggestions
