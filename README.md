# ✍️ Mad Libs - Automated Word Game

## 📝 Description
**Mad Libs** is a phrasal template word game where one player prompts others for a list of words to substitute for blanks in a story before reading it aloud. This Python program automates that process by offering three unique story templates:

* **The Hospital Stay:** A quirky story about arriving at a hospital.
* **Camping Trip:** An adventurous tale about hiking and forests.
* **Enchanted Castle:** A magical letter from a mysterious land.

---

## 🚀 Features
* **Multi-Template Support:** Users can choose from three different storylines.
* **Dynamic Story Generation:** Uses string concatenation to inject user-provided nouns, verbs, and adjectives into pre-defined strings.
* **Input Validation:** Includes an `else` block to handle invalid menu choices gracefully.

---

## 🛠️ Technical Concepts Used
* **Variables:** Storing user inputs for later use in the story.
* **Input/Output:** Using `input()` to gather data and `print()` to display the final result.
* **Conditionals:** Implementing `if-elif-else` structures to navigate between different game modes.
* **String Concatenation:** Combining static text with dynamic variables using the `+` operator.

---

## 📖 How to Play

### Prerequisites
Ensure you have **Python 3.x** installed on your system.

### Installation & Execution
1. **Clone the repository:**
   ```bash
   git clone [https://github.com/avetisyanna/mad_libs_.git](https://github.com/avetisyanna/mad_libs_.git)
   ```
2. Navigate to the directory:

```Bash
cd mad_libs_
```
3. Run the script:

```Bash
python mad_libs.py
```
### Gameplay Steps
Enter 1, 2, or 3 to choose your story theme.
Follow the prompts to enter various parts of speech (Nouns, Verbs, Adjectives).
Enjoy your hilarious custom-made story!
