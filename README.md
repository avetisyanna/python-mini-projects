# 🐍
A curated collection of Python applications, logic exercises, and automation scripts. This repository demonstrates proficiency in core programming concepts, data manipulation, and algorithmic problem-solving through a variety of hands-on projects.

---

## 🚀 Project Categories

The projects in this repository are organized by their technical focus:

### 🎮 Games & Logic
* **[Scrabble](./projects/scrabble):** A word-scoring engine that processes player data and calculates points based on letter values.
* **[Mad Libs](./projects/mad_libs):** A classic word-replacement game focusing on string manipulation and user input handling.
* **[Dice Game](./projects/dice_game):** A probability-based game utilizing the `random` module.

### 📊 Data & File Handling
* **[Hacking The Fender](./projects/hacking_the_fender):** A project focused on file I/O, specifically reading/writing CSV and JSON files to "manipulate" security data.
* **[Thread Shed](./projects/thread_shed):** A data cleaning project that takes messy string data from a sales log and transforms it into organized, usable information.

### ⚙️ Utilities & Algorithms
* **[The Boredless Tourist](./projects/boredless_tourist):** A recommendation engine that maps traveler interests to locations using Python dictionaries and list comprehension.
* **[Timer Project](./projects/timer_project):** A utility focusing on the `time` and `datetime` libraries to track intervals.

---

## 🛠️ Technical Skills Demonstrated

* **Data Structures:** Expert use of Lists, Dictionaries, Sets, and Tuples.
* **Clean Code:** Modular function design and PEP 8 naming conventions.
* **File I/O:** Reading and writing to external files (TXT, CSV, JSON).
* **Control Flow:** Advanced loops, conditional logic, and error handling (`try/except`).

## 💻 How to Use

Each project is standalone and located within its own subdirectory. To run a project:

1. Navigate to the project folder:
   ```bash
   cd projects/thread_shed
   ```
2. Run the script:
   ```bash
   python script_name.py
   ```
